This plugin display NFL teams data fetched from a remote API that is frequently updated. 

== Description ==

This plugin povide a shortcode which can be added anywhere in wordpress(posts,pages). It shows the NFL team data fectched from Remote API in two representaions.
1: Graphical
2: Tabular

You will have a setting menu for this on admin side
In the menu "NFL Teams".
From the main menu you can see setting for 

Info Header Background Color	(color picker)
Show Graphical Representation	(YES and NO) in dropdown
Shortcode                       This is just to show the shortcode need to add to page or post to see this functionality




== Installation ==

1. Upload `resulta-cc.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[NFL_TEAMS_INFO_SHORT_CODE]` in your pages or posts



== Features ==
1: Tabular data representation of dynamic fields of NFL team
    1: Sorting on all column
    2: Searching
    3:Pagination
2: Graphical data representation of dynamic fields of NFL team
    1: Option to seclect Type of graph (Pie,Column/Bar)
    2: Option to seclect dynamic field from API columns for graph
    3: Instant rendering of graph
3: Control on Table header color from admin side
4: Show/Hide control on graphical respresentaion of NFL team info data from admin.
